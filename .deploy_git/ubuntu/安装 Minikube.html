My Notes

    安装 minikube

    安装完成后,可以使用 minikube version 命令检测安装结果,正常情况下会输出版本信息

    安装 kubectl

    安装完成后,可以通过 kubectl version -o json 命令检测安装结果

    启动 Minikube (docker为驱动，常见为 VirtualBox)

    如果报错：

    解决方法：minikube start --force --driver=docker
